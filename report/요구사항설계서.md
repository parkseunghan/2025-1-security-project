# 📋 요구사항 정의서

## 1.1. 목적
- 보안이 제거된 웹 게시판을 개발하여 실습 기반 보안 위협을 체험한다.

## 1.2. 범위
- 게시글 작성/수정/삭제
- 파일 업로드/다운로드
- 회원가입/로그인/로그아웃
- 검색 기능

## 1.3. 주요 기능 목록

| 기능 | 설명 |
|------|------|
| 회원가입 | 사용자가 계정 등록 |
| 로그인/로그아웃 | 세션 생성 및 종료 |
| 게시판 작성 | 제목/내용/파일 포함 |
| 게시글 조회 | DB에서 데이터 출력 |
| 검색 | 제목 키워드로 게시글 필터링 |
| 파일 업로드 | 첨부파일 저장 |
| 파일 다운로드 | 저장된 파일 다운로드 제공 |

## 1.4. 비기능 요구사항
- DB: MySQL 5.7
- 서버: Apache 2.4 + PHP 7.4
- 컨테이너: Docker 기반 LAMP 스택
